## Overview

Bear Advanced is an elegant markdown note-taking application that organizes notes through hashtags rather than folders, paired with a clean, typographically refined writing environment that makes everyday note-taking feel considered.

Nested hashtags create a flexible organizational system where a single note can belong to multiple overlapping categories, replacing the rigid single-folder structure that most note apps impose.

Live markdown rendering shows formatted text as you type without switching between edit and preview modes, keeping the writing experience fluid while still producing properly structured markdown underneath.

A dedicated focus mode hides every interface element except the current document, narrowing the screen to just the text being written for sessions that demand sustained concentration.

For writers and note-takers who want an organizational system that adapts to how their thinking actually branches, paired with an interface that is a pleasure to write in daily, Bear Advanced is the markdown notebook built around exactly that balance.

## Performance

Bear balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
